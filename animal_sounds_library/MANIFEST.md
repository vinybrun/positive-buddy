# Animal sound library — source & license

Short animal recordings from Wikimedia Commons for choosing per-buddy
notification sounds. Most are **Public Domain / CC0**; a few priority
animals fall back to **CC-BY / CC-BY-SA** (free, but attribution
required) — the license is noted on every line so PD/CC0 can be preferred.

Priority = the 6 current buddies: **cat, dog, fox, snake, bird, butterfly**.
Butterflies are silent — see `butterfly_README.txt`.

Workflow: listen, pick a favourite per buddy, tell Claude the file + exact
crop in/out times. Chosen clips replace the placeholder tones in
`android/app/src/main/res/raw/buddy_<species>.ogg`.
